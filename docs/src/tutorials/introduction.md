# Tutorials

These tutorials provide worked examples for setting up and running atmospheric
test cases in `TrixiAtmo.jl`.

They are intended to complement the example elixirs by explaining the physical
setup, numerical choices, and expected solution behavior.

## Prerequisites

It is helpful to be familiar with the basic `Trixi.jl` workflow, including:

- equations
- meshes
- solvers
- semidiscretization
- time integration
- callbacks

For general background, see the [main Trixi.jl documentation](https://trixi-framework.github.io/TrixiDocumentation/stable/).
